A dynamic, content-managed public website for schools, showcasing activities, announcements, FAQs, admissions, results, and contact information. Fully controlled by the school administration via the dashboard.

Built with modern web technologies for a fast, responsive, and maintainable experience.

Features

Display school activities, news, and events dynamically

FAQ section for common queries

Admission form and application tracking

Contact form for inquiries

Student result display and verification

Responsive design for mobile and desktop

Real-time updates through integration with the management dashboard

Tech Stack

React + TypeScript – Interactive and type-safe frontend

Vite – Fast build tool and development server

Tailwind CSS – Utility-first styling for modern UI

Redux – State management for app-wide consistency

Axios – HTTP client for REST API calls

GraphQL – Flexible queries for fetching content

REST API – For backend communication

Getting Started

Clone the repository:

git clone <repo-url>
cd school-public-portal


Install dependencies:

npm install


Start the development server:

npm run dev
